# Wordpress boilerplate

## Quickstart

**Change the default password in `config/.env`**

```bash
# Start the containers
docker-compose up -d

# Install Deps
yarn
# Watch the JS & CSS
yarn run dev

# Go to http://localhost and activate your theme
```

## Raccomended Plugins

- Advanced Custom Fields
- All-in-One WP Migration
- Custom Post Type UI
- W3 Total Cache
- WooCommerce
- Yoast SEO
